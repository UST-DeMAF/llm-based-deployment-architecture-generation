used for evaluating the proposed EDMM transformation pipelines.

Repository:
https://github.com/open-telemetry/opentelemetry-demo

## Deployment Technology
- Ansible

## Directory Contents

- otelshopAnsible_expected.yaml: Manually curated EDMM reference model
