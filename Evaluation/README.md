# Evaluation

This folder provides all files used for the evaluation.

The folders are organized as follows:

| Folder | Purpose |
| :----- | :------ |
| Documentation | Documentation of the reference applications used in the case study |
| Measurements | All scripts and results used to calculate the metrics of the case study |
| Meitrex | Deployment model, actual EDMM model, and expected EDMM model of the Meitrex case |
| OTEL-Shop-Ansible | Deployment model, actual EDMM model, and expected EDMM model of the OTEL-Shop-Ansible case |
| OTEL-Shop-Kubernetes | Deployment model, actual EDMM model, and expected EDMM model of the OTEL-Shop-Kubernetes case |
| OTEL-Shop-Terraform | Deployment model, actual EDMM model, and expected EDMM model of the OTEL-Shop-Terraform case |
| T2Store-Microservices | Deployment model, actual EDMM model, and expected EDMM model of the T2Store-Microservices case |
| T2Store-Modulith | Deployment model, actual EDMM model, and expected EDMM model of the T2Store-Modulith case |
