# T2Store-Modulith

This directory contains the deployment artifacts of the
T2-Store Modulith application used for evaluation.

## Source Application
T2-Store Documentation

Documentation:
https://t2-documentation.readthedocs.io/en/latest/

## Deployment Technologies
- Kubernetes
- Terraform

## Directory Contents
- deploymentModel/: Original deployment artifacts
- expected.yaml: Manually curated EDMM reference model
- actual.yaml: Example generated EDMM model

## Notes
The scenario allows comparison between modulith and
microservices deployment structures under similar
deployment technologies.
